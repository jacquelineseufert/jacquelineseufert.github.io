Eurostat Projects

Over the past months I have been applying to Eurostat roles and noticed something: many of the issues raised in their own methodological papers remain unresolved in practice. Instead of waiting for an interview invitation, I decided to start building solutions myself. The projects below tackle Eurostat’s pain points directly — from reproducibility gaps to survey metadata chaos — because sometimes the best way to get attention is to fix the problems that keep getting footnoted.

I will continue to post the projects here as they progress. If any of these ideas resonate or spark interest, I would be happy to discuss them further — feel free to reach out.

1. Reproducibility Receipts

CLI tool in R/Python that fetches data by DOI, saves data + metadata, and verifies integrity via SHA-256 hashes.

Tech: Cryptographic hashing, metadata harvesting, DOI fetching.

Repository

2. LOD Mashup Gallery

Gallery of Jupyter notebooks combining Eurostat LOD with external sources (EU-LFS × Copernicus, GDP × nightlights).

Tech: SPARQL, RDF parsing, geospatial APIs.

Repository

3. NUTS Crosswalk Generator

Generates crosswalks between NUTS versions using GIS overlays and area-weighted interpolation (CSV + GeoJSON outputs).

Tech: Shapefile overlays, R (sf), Python (geopandas).

Repository

4. Eurostat in Parquet

Converter that downloads Eurostat datasets and saves them in Parquet/Arrow with schema and metadata for big-data analytics.

Tech: Data pipelines, Arrow/Parquet, Spark/Dask/Polars.

Repository

5. Survey Metadata Harvester

Scraper that collects survey documentation (variables, weights, methods) into harmonised JSON/YAML across Eurostat surveys.

Tech: Scraping, text mining, schema standardisation.

Repository

6. Break Detection QA

Automated structural break detection in Eurostat time series (Bai-Perron, CUSUM), with annotated reproducible charts.

Tech: Structural break econometrics, reproducible visualisation.

Repository

7. Vacancy Explorer

Combines Eurostat vacancy data with Online Job Ads (OJA), deduplicates ads, and maps regional labour market trends.

Tech: Web scraping, NLP deduplication, entity matching.

Repository

8. OJA Deduplication Engine

Open-source dedupe toolkit for job ads using TF-IDF, embeddings, clustering, and probabilistic record linkage.

Tech: NLP, clustering, probabilistic linkage.

Repository

9. Disclosure Control Toolkit

Toolkit implementing k-anonymity, l-diversity, and differential privacy for Eurostat-like microdata, with utility metrics.

Tech: Privacy-preserving statistics, synthetic data generation.

Repository

10. Revision Tracker

Tracks dataset revisions, computes revision statistics (bias, mean absolute revision), and visualises dataset stability.

Tech: Time series analytics, API version control, dashboards.

Repository
