Eurostat Projects

Building tools to tackle Eurostat’s methodological challenges — from reproducibility gaps to survey metadata chaos.

Over the past months I have been applying to Eurostat roles and noticed something: many of the issues raised in their own methodological papers remain unresolved in practice. Instead of waiting for an interview invitation, I decided to start building solutions myself. The projects below tackle Eurostat’s pain points directly — because sometimes the best way to get attention is to fix the problems that keep getting footnoted.

1. Reproducibility Receipts

CLI tool in R/Python that fetches data by DOI, saves data + metadata, and verifies integrity via SHA-256 hashes.

Tech: Cryptographic hashing, metadata harvesting, DOI fetching.

Repository

2. LOD Mashup Gallery

Gallery of Jupyter notebooks combining Eurostat LOD with external sources (EU-LFS × Copernicus, GDP × nightlights).

Tech: SPARQL, RDF parsing, geospatial APIs.

Repository
